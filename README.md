FC - Fucking Calculator
========================

fc is a terminal calculator with advanced operations (inspired by GNU/bc). It aims to 
be an "optimized" advanced calculator.

INSTALL
=======
First, run "./configure", then try running the generated Makefile with "make". It will warn you that it DOES NOT 
have a .config file. To fix this, use the command "make menuconfig" (although it will automatically create one 
if it doesn't detect a .config file).

(After that, it will compile and you will have the binary.)
